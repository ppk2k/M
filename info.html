Info tentang Dark Web

Dark Web adalah bagian dari World Wide Web yang tidak dapat diakses melalui browser standar seperti Google Chrome atau Mozilla Firefox. Untuk mengaksesnya, pengguna harus menggunakan perangkat lunak khusus seperti Tor (The Onion Router).

Dark Web diciptakan untuk memberikan privasi dan kebebasan berkomunikasi tanpa terdeteksi. Meskipun sering dikaitkan dengan aktivitas ilegal, Dark Web juga digunakan oleh jurnalis, aktivis, dan individu yang membutuhkan perlindungan dari pengawasan.

Dark Web ditemukan pada tahun 2000 oleh DARPA, lembaga yang mendanai penelitian untuk menjaga komunikasi aman di internet.
